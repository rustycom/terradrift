## [0.1.0] — 2026-05-16

### Added
- CLI with `scan`, `version`, `reproduce` commands.
- 17-category misconfiguration taxonomy with real-world examples.
- Drift detector with INTRODUCED / FIXED / REGRESSED events.
- Offline analyzer fallback so demos work without Checkov installed.
- Sample insecure Terraform module for `make demo`.
- GitHub Actions: `ci`, `security`, `benchmark`, `journal`, `diagrams`, `release`.
- Distroless multi-stage Dockerfile.
- ARCHITECTURE, SECURITY, THREAT_MODEL, REPRODUCIBILITY, CONTRIBUTING docs.
- LaTeX skeleton for MSR 2026 submission under `docs/paper/`.
